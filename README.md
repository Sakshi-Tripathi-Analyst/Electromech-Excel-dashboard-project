# 📈 Electromech Sales Dashboard (Excel)

## 📌 Project Overview
This project is an Excel-based dashboard designed to analyze and visualize sales data for Electromech. It helps track performance, trends, and key business metrics.

## 🚀 Key Features
- Interactive Excel dashboard
- Sales performance tracking
- Dynamic charts and pivot tables
- KPI monitoring (Revenue, Growth, Orders)

## 📊 Dashboard Insights
- Monthly sales trends and growth patterns
- Region-wise and product-wise performance
- Identification of high-performing segments

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables & Charts
- Data Cleaning
